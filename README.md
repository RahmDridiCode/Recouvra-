# Recouvra+

Backend Node.js pour la gestion du recouvrement des factures impayées.

Tech stack:
- Node.js 22
- Express
- MongoDB + Mongoose
- JWT
- Joi
- Swagger
- Jest

Scripts:
- `npm run dev` : démarre en dev avec nodemon
- `npm start` : démarre la production
- `npm test` : lance les tests

Voir `.env.example` pour les variables d'environnement.
